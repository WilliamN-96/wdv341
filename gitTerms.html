Terms for Git

  • Verstion Control Software: A system that records changes to a file or set of files over time so that you can recal a specific verions later.
  • Add: A command that adds a change in the working directory to the staging area.
  • Commit: Saves changes to the local repository.
  • Push: Used to upload a local repository to a remotr repository.
  • Pull: Used to get and download content from a remote repository and immediately update the local repositry to match the content.
  • Clone: Takes and existing repo and creates a clone/copy in a new directory.
